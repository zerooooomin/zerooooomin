
컬렉션의 종류

| 종류(인터페이스)  | 개요                                                           |
| :--------- | :----------------------------------------------------------- |
| 리스트 (List) | 순서대로 데이터를 저장. 데이터의 중복을 허용                                    |
| 세트 (Set)   | 순서를 유지하지 않는 집합. 중복을 허용하지 않음                                  |
| 맵 (Map)    | 키와 값의 쌍으로 이루어진 데이터 집합. 순서는 유지되지 않고 키의 중복을 허용하지 않으며 값의 중복을 허용 |
